FBX Converter preferences

 
 
 

The FBX Converter preferences window lets you set general options. The Preferences window has three tabs:

  • General
  • FBX Explorer
  • FBX Viewer

General preferences

Use the FBX Converter preferences to specify general settings for the FBX Converter application, such as whether the current layout is saved and used at next launch and the preferred file path for downloaded FBX files.

FBX Explorer preferences

Use the FBX Explorer preferences to specify things such as data display maximums, files to exclude, and file types for comparison.

See Changing explorer preferences for more.

FBX Viewer preferences

Use the FBX Viewer preferences to specify whether files start playing immediately on load or whether instances of the FBX Viewer should close on exit.

See Changing viewer preferences for more.
