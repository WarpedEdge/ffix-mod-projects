# CharacterPack

Notes:
- Main mod for gerneric items everyone will use in `CharacterPack/`.
- Submods are in `CharacterPack/Characters/<Name>/` and have character specific related items in `StreamingAssets/`, `FF9_Data/`, and `DictionaryPatch.txt`.
- `StreamingAssets/` is for data, scripts, SFX, models, and animations.
- `FF9_Data/` is for UI atlases, localized text, and custom textures.

## Explanations (folders and main files)

### Main

- `CharacterPack.md` - this layout guide.
- `ModDescription.xml` - Memoria mod metadata and submod list.
- `Characters/` - per-character submod (Cloud is the only one right now).
- `FF9_Data/` - main mod UI atlases, text, and custom textures.
- `StreamingAssets/` - main mod data, scripts, and SFX.

### Characters/Cloud

- `DictionaryPatch.txt` - Cloud-specific text and string overrides.
- `StreamingAssets/` - Cloud data, animations, models, and SFX.
- `FF9_Data/` - Cloud-specific UI atlases and custom textures.

### Characters/Cloud/StreamingAssets/Assets/Resources

- `Animations/` - all Cloud animation resources and model viewer animation folders.
- `Models/2/11000..11024/` - Cloud models (including costumes) and their textures.
- `battlemap/battlemodel/6/12000..12007/` - Cloud weapon models and textures.

### Characters/Cloud/StreamingAssets/Data

#### Battle

- `Actions.csv` - Cloud's abilities
- `StatusData.csv` - Cloud-specific status' (modes and beam echo)

#### Characters

- `Abilities/58.csv` - Cloud's ability list.
- `Abilities/AbilityFeatures.txt` - Cloud's ability feature sets and command swaps.
- `Abilities/AbilityGems.csv` - Cloud's ability gem costs.
- `BaseStats.csv` - Cloud's base stat templates.
- `BattleParameters.csv` - Cloud's battle model/anim parameters.
- `CharacterParameters.csv` - Cloud's character parameters and flags.
- `CommandSets.csv` - Cloud's command layout.
- `Commands.csv` - Cloud's command IDs.
- `DefaultEquipment.csv` - Cloud's starting equipment.

#### Items

- `ItemEquipPatch.txt` - Cloud's equipment flags.
- `ItemShopPatch.txt` - Cloud's shop list additions (This is so his items do not show up when he is disabled for modularity future additions to this mod).
- `Items.csv` - Cloud's items.
- `Stats.csv` - Cloud's Weapon stat sets.
- `Weapons.csv` - Cloud's weapons.

#### SpecialEffects

- `Common/SHP.csv` - Cloud's SHP icon setup for status sprites.
- `ef11000..ef11019/PlayerSequence.seq` - per-ability SFX sequences.

### Characters/Cloud/FF9_Data

- `CustomTextures/` - Cloud's stance/status icons (Operator/Prime/Punisher/BeamEcho).
- `embeddedasset/UI/Atlas/` - Cloud's UI atlases (Face/Icon/TutorialUI).

### Main mod FF9_Data

- `CustomTextures/` - main mod custom textures (status/SHP/UI).
- `embeddedasset/text/` - main mod localized text overrides.
- `embeddedasset/UI/Atlas/` - main mod UI atlases (Face/Icon).

### Main mod StreamingAssets

- `Data/` - main mod data (Battle, Characters, Items, SpecialEffects).
- `Scripts/Sources/` - main mod C# script locations.
- `Scripts/` (compiled) - compiled Memoria script assemblies.
