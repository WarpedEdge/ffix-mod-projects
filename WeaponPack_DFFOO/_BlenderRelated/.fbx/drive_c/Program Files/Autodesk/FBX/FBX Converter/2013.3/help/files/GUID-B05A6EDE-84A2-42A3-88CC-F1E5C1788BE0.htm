Comparing files

 
 
 

To compare files:

  1. Launch two FBX Explorers.

    Two FBX Explorers A, B.

  2. Load an FBX file in each FBX Explorer.
  3. Click Compare Structure, Compare Values, or Compare All.

    FBX Explorer: A. Compare buttons

Compare structure

Lets you contrast differences between data structures.

Compare values

Lets you find differences between data values.

Compare all

Lets you find differences in both data structure and values.
