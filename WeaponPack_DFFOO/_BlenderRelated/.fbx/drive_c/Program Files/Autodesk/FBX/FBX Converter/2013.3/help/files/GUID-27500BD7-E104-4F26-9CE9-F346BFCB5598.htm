Default FBX options

 
 
 

When the FBX Converter performs an automatic conversion, such as when you drag a non-native (OBJ/COLLADA DAE/DXF/3DS) file into the FBX Viewer defaults are applied.

The following are default FBX conversion settings:

Format Converted by default
3DS
  • Textures
  • Materials
  • Animation
  • Mesh
  • Lights
  • Cameras
  • Convert Units
  • Split Normals
DXF
  • Add Root node
  • Weld vertices
  • Derive primitives by layer
OBJ
  • Add Root node
DAE n/a
All
  • Embed media
  • Save as Binary
NoteThese defaults are the same as the FBX Plug-in Autodesk Media & Entertainment preset.
